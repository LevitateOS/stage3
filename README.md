# stage3

Stage3 tarball builder for LevitateOS.

## Overview

Builds stage3 tarballs containing the base system that gets extracted during LevitateOS installation.

## Building

```bash
cargo build
```

## Usage

```bash
cargo run
```

## Development

```bash
# Build
cargo build

# Run tests
cargo test

# Check with clippy
cargo clippy
```
